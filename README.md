### Используемые библиотеки:
aiosqlite(0.21.0), aiogram(3.20.0), aiohttp(3.11.18)

python-dotenv(1.1.0) для работы с секретными ключами

### Реализованные команды:
- /start, /help  -  показ приветственного сообщения и информации.

- /stats - показывает статистику(топ-5) популярных запросов боту.

- /history - показывает историю последних 10 запросов.

- /clear - очищает историю запросов локально.

### Описание работы:
- Бот создаёт сессию в начале работы и инициализирует базу данных,

- Обрабатывает каждый запрос пользователя как команду из списка, или поисковой запрос.

- При поисковом запросе через единственную `ClientSession` отправляет GET-запрос к `https://kinopoiskapiunofficial.tech/api/v2.1/films/search-by-keyword`.  
Если ответ содержит хотя бы один фильм, берётся **первый (самый релевантный)**, и делается **второй запрос** за подробной информацией (`/films/{filmId}`).

- Достаёт всю подробную информацию о фильме, (постер, название, страна, год выпуск, описание, длительность, рейтинг)

- Достаёт ссылку на бесплатный просмотр, обрезает до ограничений описание при необходимости и отправляет пользователю

- Запрос сохраняется в базу данных

- Выдаётся достижение за 5 запросов:)# tg_sinemabot
